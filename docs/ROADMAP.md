# Implementation Roadmap

## Phase 1: Foundation (Weeks 1-2)
- Scaffold repo, templates, CI, and coding standards.

## Phase 2: Data Ingestion (Weeks 3-6)
- Build eTender, municipal, and gazette connectors.
- Schedule periodic fetch jobs.

## Phase 3: ETL and OCR (Weeks 5-10)
- Normalize source records into a single contracts schema.
- Add OCR path for scanned PDFs.

## Phase 4: Features and Modeling (Weeks 7-14)
- Engineer risk features and baseline anomaly model.
- Add entity resolution and network clustering.

## Phase 5: Verification and Evidence (Weeks 12-18)
- Deliver review UI and evidence packaging workflow.

## Phase 6: Hardening and Deployment (Weeks 15-24)
- Increase test depth, add security scans, deploy in SA region.
